## [1.0.0] - 2024-12-XX

### 🎉 首次发布
这是 AgenticSqlConverter 的第一个正式版本，包含完整的核心功能和架构。

### ✨ 新增功能
- **多代理架构**: 实现智能 SQL 转换的核心架构
  - CTE 提取代理
  - LLM 翻译代理
  - SQL 验证代理
  - 语法修复代理
  - SQL 组装代理
  - 优化代理

- **SQL 方言支持**:
  - PostgreSQL 到 Databricks SQL 转换
  - 可扩展的方言支持框架

- **LLM 集成**:
  - AI infrastructure GPT 系列支持
  - Anthropic Claude 支持
  - 可插拔的 LLM 提供者架构

- **智能解析**:
  - CTE (Common Table Expression) 提取
  - SQL 分块策略
  - 依赖关系分析
  - 语法树分析

- **转换引擎**:

## 📋 版本说明

### 版本号格式
- **主版本号**: 不兼容的 API 变更
- **次版本号**: 向后兼容的功能新增
- **修订版本号**: 向后兼容的问题修复

### 发布周期
- **主版本**: 重大功能发布，每 6-12 个月
- **次版本**: 功能增强，每 2-4 个月
- **修订版本**: 问题修复，每 1-2 周

### 支持策略
- **最新版本**: 完全支持，包含所有功能和安全修复
- **LTS 版本**: 长期支持版本，包含安全修复
- **旧版本**: 有限支持，仅包含严重问题修复
